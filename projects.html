Projects

Distributed Inference with Limited Communication (Ph.D. Work)

This work focuses on developing communication-efficient algorithms for distributed systems, particularly in federated learning settings. The goal is to analyze the trade-off between estimation accuracy and communication cost across multiple clients.

The research involves designing and evaluating statistical inference algorithms under bandwidth constraints, along with developing quantization techniques for efficient transmission of information in distributed environments.

Mean Estimation in Federated Learning Systems

Implemented and analyzed algorithms for distributed mean estimation in federated learning under homogeneous and heterogeneous communication conditions. The work focused on robustness to packet loss and varying communication budgets.

The system was implemented using the Flower Framework, demonstrating improved performance over existing techniques across diverse network scenarios.

Communication-Efficient Distributed Learning Systems

Explored various quantization and compression techniques for distributed learning systems, focusing on reducing communication overhead while maintaining model performance.

This work integrates concepts from distributed optimization, multiple access communication, and signal processing to improve scalability in federated environments.

Data Security for IoT Networks (M.Tech Project)

Developed a secure data encryption framework using Cellular Automata for IoT-based healthcare systems. The work involved implementing encryption and decryption techniques using rule-based cellular automata.

The system was tested on real-world healthcare data, demonstrating effective security performance with low computational complexity.
